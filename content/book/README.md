# Book Content Storage

This directory contains the full book content organized as markdown files.

## Structure

Organize your book content here by section and/or chapter. Suggested structure:

```
content/book/
├── section-01/
│   ├── chapter-01.md
│   ├── chapter-02.md
│   └── ...
├── section-02/
│   ├── chapter-05.md
│   └── ...
└── ...
```

Or if your book doesn't have sections:

```
content/book/
├── chapter-01.md
├── chapter-02.md
├── chapter-03.md
└── ...
```

## File Naming

- Use descriptive filenames: `chapter-01.md`, `chapter-02.md`, etc.
- Or use chapter titles as slugs: `the-test.md`, `in-the-poo.md`
- Keep it consistent

## Markdown Format

Each chapter markdown file should contain:
- The full chapter content
- Use standard markdown formatting
- Headings, paragraphs, quotes, etc.
