⚙️ GitHub Auto-Save Setup

To enable automatic saving (no manual uploads needed):

  1. Create a GitHub repository for your site
  2. Create a Personal Access Token with repo permissions
  3. Go to the "Settings" tab below and configure GitHub integration

Without GitHub: You'll download JSON files and upload them manually to your server.

GitHub Auto-Save Setup

When enabled, changes save directly to GitHub and auto-deploy.
Format: your-username/your-repo-name
Usually "main" or "master"
How to create:
1. Go to GitHub.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token with repo permissions
3. Copy token and paste here (keep it secret!)

Note: Token is stored locally in your browser. Never share it publicly.
